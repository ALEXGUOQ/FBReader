Book Tags in FBReader

What is a book tag?

As of version 0.8.16 FBReader supports book tags. This feature allows for organizing libraries in a way more convenient for the end user.

A tag is just a text string, set of possible tags is not pre-defined, which means that the user can add a new tag at any moment. Each book can have one or more tags. These tags are listed in the book information dialog as a list.

For example, if the dialog shows «Science Fiction, Fantasy, New», it means that the book has three tags — «Science Fiction», «Fantasy» and «New».

Tag name must not contain a comma symbol («,»).

Spaces and tab symbols at the beginning and the end of a tag name are ignored.

Hierarchical tags.

If tag name contains «/», then this tag is considered hierarchical. For example, tag list «Fiction/SF & Fantasy/Fantasy» will be interpreted as subtag «Fantasy» of subtag «SF & Fantasy» of top level tag «Fiction».

Tag name must not contain a slash symbol («/»).

Library Tree View.

Book library is shown as a tree. First levels of the tree are representing book tags (one or more), next levels are for the book author, for the book series (if present) and the book title correspondently.

If a book has several tags, it will be available under each tag it has.

If a book has no tags, it will be shown in the branch called «Books with no tags».

Besides, all books can be found in the branch called «All Books».

Library Tree View Settings.

If you open the preferences dialog and select «Library» tab, you can see two options:

«Sort Books by Tags»
By clearing this option you tell FBReader to show traditional tree with not tags

By clearing this option you tell FBReader to not show the branch called «All Books»

Book Highlighting in the Library View Tree.

When you open library tree view, the book you are currently reading will be shown and highlighted in the tree. While all locations where the book is available are highlighted, you will see the last appearance of the book in the view. You can go through all the locations using «Find Next»/«Find Previous» buttons.

How Tags are Defined.

All tags are defined by the user. Following sections describe this in more details. Besides, certain book formats support tags natively. As of version 0.8.16, FBReader can extract tags from books in fb2 and oeb/epub formats.

Settings/Changing Tags for a Book.

In order to set or to change list of tags for a book, open book information dialog, select «Tags» tab and edit the list. Tag names will only be changed for the book you update. If you add a new tag, it will be automatically shown in the tree. If you delete a tag that only this book had, the tag will be automatically removed from the tree.

Changing/Merging/Copying Tags.

If you would like to change a tag, you can do it in the library tree view: navigate to the tag you'd like to change and click on the bulb that is to the right to the tag name.

In the opened dialog you can:

  • Change the tag name

    • NB: a tag name must not contain a comma: if you put it there by mistake, the name won't be changed.

    Empty name won't be changed either.

    If you use «/» in the tag name, it will be considered as a separator for a hierarchical tag.

    Spaces and tab symbols will removed from the beginning and the end of the tag name.

  • Merge the tag with an existing one

    • To merge the tag with another one, you just need to change the name of the latter or simply select it from the list of available tags.

  • Cloning a tag.

    • Check the box «Create clone» and specify the name of the new tag. If the tag you specified already exists, the books from with the current tags will be tag with the new tag.

    Unlike others, this operation can also be performed for the tags «All Books» and «Books with no Tags».

Deleting Tags.

If you would like to delete a tag, click on a minus sign right to the tag name. You will need to confirm the operation when a dilog appears. If you confirm the operations, the specified tag will be removed from all books you have. Note: no books will be removed by this operation.

If the tag you are about to delete has subtags, the confirmation dialog will have three buttons — «This Only», «With Subtags« and «Cancel». If you click on the first one, exactly the selected tag will be removed (if there are any books tagged by it). If you click on the second one, then the selected tag and all its subtags will be removed and the corresponding branch will not be shown in the tree.

For example, if you have books tagged as «Fiction» and books tagged as «Fiction/Horror», and you try to delete tag «Fiction;», you can

  • Click on «This Only»

    • In this case, the tag «Fiction» will disappear from the tag list of books that were tagged with it, while tag «Fiction/Horror» will remain. And while both tags will be shown in the library tree view, the first one will have no books.

  • Click on «With Subtags»

    • In this case both tags «Fiction» and «Fiction/Horror» will be removed from the books that were tagged with them and the whole subtree that started with «Fiction» will not be shown.
